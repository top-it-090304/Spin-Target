# Визуальные эффекты

## Шлейф ножа (Trail)

**Файл:** `elements/knife/knife.gd`, `elements/knife/knife_trail_mote.gd`

Во время полёта нож оставляет шлейф из двух слоёв:

### Линейный шлейф (Trail2D)
- Непрерывная лента, следующая за ножом.
- Ширина: 2.0–6.0 пкс в зависимости от типа.
- Длина: 32–54 пкс.
- Прозрачность (альфа): 0.24–0.52.
- Цвет совпадает с `effect_color` ножа.

### Мотки (Motes)
Маленькие круги, отрывающиеся от шлейфа:
- Радиус: 2.4–4.0 пкс.
- Живут: 0.16–0.22 с.
- Цвет: `effect_color` ножа.
- Спавн-интервал: 0.06–0.1 с (зависит от типа ножа).
- Некоторые типы ножей (1, 4) вообще не имеют моток.

## Вспышка при попадании

**Файл:** `elements/targets/target/target.gd`

При ударе ножа о дерево мишени:

### Вспышка (Flash)
- Текстура: круглый градиент 56×56 пкс.
- Цвет: `effect_color` текущего ножа.
- Прозрачность: 0.85.
- Длительность: 0.12 с.
- Масштаб: определяется весом ножа и типом эффекта.

### Расходящиеся линии (Hit marks)
- 3–7 линий (зависит от `hit_effect_kind`).
- Ширина: 2.5–4.0 пкс.
- Расходятся от точки удара.
- Затухают за 0.16 с.

### Масштаб вспышки по типу эффекта

| Эффект | Масштаб | Линий |
|--------|--------|-------|
| 0, 3 | +18% от веса | 4 |
| 1, 4, 7 | +18% от веса | — |
| 2, 5 | +28% от веса | 7 |
| 6, 8 | +24% от веса | 5 |

## Тряска камеры

**Файл:** `elements/targets/target/target.gd`

Активируется если `weight_multiplier` ножа ≥ 1.2:

```
strength = 5.0 + 5.0 × (weight - 1.2)
duration = 0.1 + 0.06 × (weight - 1.2)
decay    = 26.0 единиц/с
```

**Пример для Тяжёлого ножа (вес 1.65):**
- Сила: 5.0 + 5.0 × 0.45 = **7.25**
- Длительность: 0.1 + 0.06 × 0.45 ≈ **0.127 с**

## Частицы уничтожения яблока

**Файл:** `elements/apple/apple.gd`

При сборе яблока запускаются 2 системы частиц:

- Начальная скорость: 150–200 пкс/с.
- Угловая скорость частиц: −400 до +400 °/с.
- Для золотого яблока: частицы с золотистым оттенком.
- Длительность: одноразовые (one-shot), удаляются по завершению.

---

## Частицы уничтожения мишени

**Файл:** `elements/targets/target/target.gd`

После победы на уровне (все яблоки собраны) мишень разрушается:

- 3 системы частиц, специфичных для каждого уровня (уникальные текстуры).
- Длительность: 1.0 с.
- Угол разброса: 180°.
- Скорость: 150–250 пкс/с.
- Гравитация: 400 пкс/с².

---

## Шейдер золотого яблока

**Файл:** `elements/apple/apple.gd` + кастомный шейдер

Золотое яблоко визуально отличается:
- Золотая перекраска спрайта (кастомный шейдер).
- Пульсирующая аура вокруг яблока.
- Анимация ауры продолжается пока яблоко висит на мишени.

## Плавающий текст награды

**Файл:** `elements/targets/target/apple_reward_floater.gd`

Появляется над точкой попадания при сборе яблока.

### Параметры анимации

| Параметр | Значение |
|---------|---------|
| Размер шрифта | 80 пкс |
| Длительность | 1.05 с |
| Подъём | 50 пкс/с |
| Поп-анимация | 0.7→1.18 за 0.12 с, затем 1.0 |
| Затухание альфы | Квадратичное |

### Цветовая схема

| Тип попадания | Текст | Цвет |
|--------------|-------|------|
| Обычное яблоко | `+X` | `(255, 235, 72)` — золотой |
| Мульти-удар (2) | `Метко! +X` | `(255, 184, 31)` — тёмно-золотой |
| Мульти-удар (3+) | `Метко! ×N +X` | `(255, 184, 31)` — тёмно-золотой |

## HUD-индикаторы

### Иконки ножей (нижняя часть экрана)
- 7 иконок ножей.
- Использованные ножи визуально затухают (fade out).
- Доступный нож отображается ярко.

### Иконки уровней (верхняя часть экрана)
- 6 иконок, каждая соответствует одному уровню.
- Текущий уровень подсвечен, пройденные — закрашены.
- Визуально уровень-босс отличается от остальных.
