# Комбо и система наград

## Комбо

**Файл:** `core/globals.gd`

### Как работает
- Каждое успешное попадание ножа в мишень (дерево или яблоко) прибавляет +1 к комбо.
- Комбо **сбрасывается** при столкновении с другим ножом, при промахе мимо мишени, при условии поражения.
- Комбо **не сбрасывается** между уровнями — продолжается на протяжении всей игровой сессии.

### Мультипликатор комбо

| Комбо | Множитель |
|-------|----------|
| 0 | 1.0× |
| 1 | 1.0× |
| 2 | 1.1× |
| 3 | 1.2× |
| … | … |
| 21 | 3.0× (максимум) |

**Формула:** `min(1.0 + (combo - 1) * 0.1, 3.0)`

## Мульти-удар (Меткий удар)

**Файл:** `elements/targets/target/target.gd`

### Условие активации
Один нож сбивает **2 и более яблок** за один бросок. Окно регистрации — **0.12 секунды** между последовательными попаданиями.

### Награда при мульти-ударе
```
multiplier = 5 × knife.sharp_hit_multiplier
```

- Обычный нож: 5×
- Нож «Меткий»: 5 × 1.4 = **7×**

### Отображение
- При 2 яблоках: **«Метко! +X»** (оранжевый текст)
- При 3+ яблоках: **«Метко! ×N +X»** (с указанием кратности)
- Цвет: `(255, 184, 31)` — тёмно-золотой

---

## Формула итоговой награды за яблоко

```
reward = base_reward
       × knife.apple_reward_multiplier   (если обычное яблоко)
       × knife.golden_reward_multiplier  (если золотое яблоко)
       × combo_multiplier
```

Для **золотого яблока** базовая награда уже ×10:
```
golden_reward = 1 × 10 × golden_reward_multiplier × combo_multiplier
```

На уровне босса (5) базовая награда за яблоко:
```
boss_apple_reward = boss_index × 10
```
где `boss_index = round_number + 1` (номер прохождения данного раунда).

---

## Формула итогового счёта за забег

```
score = level × 10000
      + apples × 20
      + sharp_hits × 500
      + golden_apples × 700
```

| Компонент | Вес |
|-----------|-----|
| Уровень (текущий) | 10 000 за уровень |
| Яблоки | 20 за яблоко |
| Метких ударов | 500 за удар |
| Золотых яблок | 700 за яблоко |

## Визуализация наград

**Файл:** `elements/targets/target/apple_reward_floater.gd`

Плавающий текст появляется над точкой попадания:

| Тип | Текст | Цвет |
|-----|-------|------|
| Обычное яблоко | `+X` | Золотой `(255, 235, 72)` |
| Мульти-удар (2 яблока) | `Метко! +X` | Тёмно-золотой |
| Мульти-удар (3+ яблок) | `Метко! ×N +X` | Тёмно-золотой |

### Анимация плавающего текста
- **Поп:** масштаб 0.7 → 1.18 за 0.12 с, затем обратно до 1.0.
- **Подъём:** 50 пкс/с вверх.
- **Затухание:** квадратичное по альфе за 1.05 с.
- Размер шрифта: 80 пкс.
