# Spin Target — Документация игры

## О проекте

**Название:** Spin Target (Аврора)  
**Жанр:** Аркада / One-Hit  
**Движок:** Godot 4  
**Платформа:** Aurora OS  
**Язык:** GDScript  
**Разрешение:** 540 × 960 пикселей (вертикальная ориентация)

## Суть игры

Игрок бросает ножи в крутящуюся мишень, стараясь попасть по всем яблокам, пока не закончились ножи. Нож может:
- попасть в яблоко — хорошо, яблоко уничтожается, начисляются очки;
- воткнуться в дерево мишени — нож остаётся торчать, мишень продолжает вращаться;
- попасть в уже воткнутый нож — промах, нож падает, комбо сбрасывается.
