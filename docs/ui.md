# Интерфейс (UI)

## Стартовый экран

**Файл:** `scens/start_screen/start_screen.gd`

### Элементы
- **Превью текущего ножа** — показывает выбранный нож с названием.
- **Кнопка «Играть»** — запускает игровую сцену.
- **Кнопка «Магазин»** — переход в магазин ножей.
- **Кнопка «Настройки»** — открывает оверлей настроек.
- **Карточки лучшего забега** — 4 карточки с лучшим результатом из топ-5.

### Карточки лучшего результата

| Карточка | Данные | Цвет |
|---------|--------|------|
| 1 | Уровень | Голубой |
| 2 | Яблоки | Белый |
| 3 | Метких ударов | Фиолетовый |
| 4 | Золотых яблок | Голубой |

---

## Оверлей настроек

Открывается поверх стартового экрана:
- **Слайдер громкости музыки** (0–100%).
- **Кнопка «Сбросить прогресс»** — открывает модальное окно подтверждения.
- **Закрытие:** кнопка назад или клавиша Escape.

---

## Диалог выхода

При нажатии системной кнопки «Назад» на стартовом экране:
- Модальное окно с двумя кнопками: **«Остаться»** / **«Выйти»**.

## HUD (во время игры)

**Файл:** `elements/ui/hud/hud.gd`

### Компоненты HUD

| Элемент | Расположение | Описание |
|---------|-------------|----------|
| Счётчик яблок | Верх-правый угол | Баланс яблок, обновляется по сигналу |
| Иконки ножей | Низ-левый угол | 7 иконок, использованные затухают |
| Иконки уровней | Верх | 6 иконок прогресса по уровням |
| Кнопка «В меню» | Фиксированная | Контекстная: «ВЫХОД» или «В МЕНЮ» |

HUD обновляется каждый кадр для синхронизации с игровым состоянием.

## Баннер победы

**Файл:** `elements/ui/win_banner/win_banner.gd`

Показывается после сбора всех яблок на уровне.

### Содержимое

| Поле | Обычный уровень | Босс |
|------|----------------|------|
| Заголовок | «Пройдено» | «Победа» |
| Статистика | 4 карточки с итогами | 4 карточки с итогами |

### Карточки результатов уровня

| Карточка | Данные | Цвет |
|---------|--------|------|
| 1 | Уровень | Голубой |
| 2 | Яблоки за уровень | Белый |
| 3 | Метких ударов | Фиолетовый |
| 4 | Золотых яблок | Голубой |

### Поведение
- Отображается **2.8 секунды**.
- Анимация подгонки шрифта (font-fit).
- По истечении автоматически запускает следующий уровень.

## Баннер поражения

**Файл:** `elements/ui/restart_overlay/restart_overlay.gd`

Показывается когда все 7 ножей брошены, а яблоки остались.

- Текст: **«ПОРАЖЕНИЕ»**.
- Длительность: ~1.5 секунды.
- Действия при показе:
  1. Сохранение статистики забега.
  2. Сброс уровня до 0.
  3. Инициализация нового забега.
  4. Возврат на стартовый экран.

## Магазин ножей

**Файл:** `scens/knife_shop/knife_shop.gd`

### Структура

```
Магазин
├── Заголовок «CUSTOMIZE»
├── Строка превью
│   ├── Панель с изображением ножа
│   └── Панель с описанием
│       ├── Название ножа
│       ├── Описание
│       └── 4 строки характеристик
├── Разделитель
├── Сетка 3×3 (9 ножей)
└── Кнопка разблокировки
```

### Панель характеристик

4 характеристики, каждая со шкалой из 5 делений (pip-система):

| Параметр | Цвет | Описание при максимуме |
|---------|------|----------------------|
| Скорость | Синий | «очень высокая» при ≥ 1.18× |
| Лезвие | Золотой | «широкий» при ≥ 1.28× |
| Вес | Оранжевый | «тяжёлый» при ≥ 1.45× |
| Бонус | Зелёный | Активные множители в % |

### Кнопка разблокировки
- Показывает стоимость следующей разблокировки в яблоках.
- При нажатии — случайный незаблокированный нож открывается.
- Если все ножи уже открыты — кнопка недоступна.

### Элемент ножа в сетке (ShopItem)
- Показывает спрайт ножа.
- Заблокированные ножи отображаются с затемнением и иконкой замка.
- Клик по разблокированному — экипирует нож (сохраняется).
- Клик по заблокированному — показывает стоимость.
