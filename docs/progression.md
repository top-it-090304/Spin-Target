# Прогрессия и прохождение

## Система уровней

Всего **6 уровней** (индексы 0–5). После победы на уровне 5 (Босс) цикл сбрасывается до 0, но сложность нарастает.

| Уровень | Характер вращения | Особенность |
|---------|------------------|-------------|
| 0 | Постоянное, одна сторона | Начальный |
| 1 | + флуктуации скорости | Каждые 2 с: ×0.6–1.8 |
| 2 | Быстрее + флуктуации | — |
| 3 | Ещё быстрее + флуктуации | — |
| 4 | Чередование направлений | 5 с по часовой / 3 с против |
| 5 | Непредсказуемое | 0.5–2.0 π рад/с, смена каждые 1–3 с |

---

## Босс (Уровень 5)

### Механика
- Вращение полностью случайное: скорость и направление меняются независимо.
- Психологически давит — нельзя угадать паттерн.

### Награда за победу над боссом
Начисляется **только если побит предыдущий рекорд**:

```
knife_factor = (current_knife_index + 1) × 0.2
step         = 0.2 × knife_factor
bonus        = round(10 + step × total_levels_passed)
```

**Пример:** нож #3, пройдено 20 уровней:
```
knife_factor = (3 + 1) × 0.2 = 0.8
step         = 0.2 × 0.8    = 0.16
bonus        = 10 + 0.16 × 20 = 13.2 → 13 яблок
```

Более дорогой нож и больше пройденных уровней = большая награда.

## Статистика забега

Каждый забег отслеживает:

| Параметр | Описание |
|---------|----------|
| `run_apples_gained` | Всего яблок заработано |
| `run_throws` | Всего бросков |
| `run_sharp_hits` | Сколько мульти-ударов |
| `run_golden_apples` | Сколько золотых яблок собрано |
| `run_levels_completed` | Пройдено уровней за забег |
| `run_max_level` | Максимальный уровень за забег |

---

## Лучшие результаты

Система хранит **топ-5 забегов** по очкам.

Данные каждого сохранённого забега:
- Уровень
- Пройдено уровней
- Яблоки
- Метких ударов
- Золотых яблок
- Бросков
- Итоговый счёт

Отображаются на стартовом экране в виде 4 карточек (лучший из топ-5).

## Глобальная прогрессия

| Параметр | Описание |
|---------|----------|
| `total_levels_passed` | Всего пройдено уровней за всё время |
| `max_level_record` | Максимальное число последовательно пройденных уровней + победы над боссом |
| `apples` | Текущий баланс яблок (валюта) |
| `current_knife_index` | Выбранный нож |
| `unlocked_knives` | Массив bool[9] — какие ножи разблокированы |

---

## Система сохранений

**Файл сохранения:** `user://save.cfg`

```ini
[progress]
apples              = <int>
current_level       = <int>
current_knife_index = <int>
unlocked_knives     = <bool array>
total_levels_passed = <int>
max_level_record    = <int>

[records]
best_runs = <array of dicts>

[settings]
music_volume = <float 0.0–1.0>
sfx_volume   = <float 0.0–1.0>
```

Автосохранение происходит после каждого значимого действия: покупки, завершения уровня, поражения.

---

## Яблоки как валюта

Яблоки — единственная игровая валюта. Тратятся на разблокировку ножей в магазине.  
Зарабатываются во время игры (за попадания, комбо, мульти-удары, золотые яблоки).  
Баланс сохраняется между сессиями.
