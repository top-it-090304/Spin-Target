# Spin Target

**Spin Target** - учебная аркадная игра на Godot 4. Игрок бросает ножи во вращающуюся мишень, старается попадать по яблокам, зарабатывать очки и проходить уровни, не попадая в уже воткнутые ножи. В проекте есть магазин ножей, разные характеристики оружия, комбо за точные попадания, боссы, визуальные эффекты, музыка и сохранение прогресса. Игра ориентирована на мобильный вертикальный экран и подготовлена для демонстрационной сборки под ОС Аврора и Android.

## Как скачать и запустить готовую версию

Для обычного запуска собирать проект не нужно. Готовый RPM-пакет опубликован в GitHub Releases.

1. Откройте релиз [demo-2026-05-18](https://github.com/top-it-090304/Spin-Target/releases/tag/demo-2026-05-18).
2. Скачайте файл `SpinTarget-demo-2026-05-18-AuroraOS.rpm`.
3. Перенесите RPM на устройство или эмулятор с ОС Аврора.
4. Установите пакет штатным способом.
5. Запустите игру из списка приложений.

Скачать RPM из терминала можно так:

```bash
curl -L -o SpinTarget-demo-2026-05-18-AuroraOS.rpm \
  https://github.com/top-it-090304/Spin-Target/releases/download/demo-2026-05-18/SpinTarget-demo-2026-05-18-AuroraOS.rpm
```

Для установки RPM на устройстве с ОС Аврора можно использовать стандартный установщик пакетов. Если установка выполняется из терминала в режиме разработчика, используйте локальную установку RPM:

```bash
devel-su pkcon install-local SpinTarget-demo-2026-05-18-AuroraOS.rpm
```

После установки игра появляется в меню приложений и запускается как обычное приложение.

## Как запустить исходники из терминала

Этот способ нужен разработчикам, если требуется открыть проект из исходного кода, а не запускать готовый RPM.

1. Установите **Godot Engine 4.4.1** или совместимую версию Godot 4.4.
2. Склонируйте репозиторий:

```bash
git clone https://github.com/top-it-090304/Spin-Target.git
cd Spin-Target
```

3. Запустите проект:

```bash
godot --path spin-target
```

Также можно открыть `spin-target/project.godot` в редакторе Godot и нажать **Run Project**.

## Как собрать из исходников

Самостоятельная сборка нужна только разработчикам. Для проверки демо достаточно скачать RPM из релиза.

Сборки выполняются из проекта `spin-target/project.godot`. Основные настройки экспорта лежат в файле `spin-target/export_presets.cfg`.

### Android APK

1. Установите Godot 4.4.1.
2. Установите Android SDK, Android Build Tools и JDK.
3. В Godot откройте **Editor Settings -> Export -> Android** и укажите пути к Android SDK и JDK.
4. Откройте проект `spin-target/project.godot`.
5. Перейдите в **Project -> Export**.
6. Выберите пресет **Android**.
7. Нажмите **Export Project** и сохраните APK-файл.

Если Godot доступен из командной строки, APK можно собрать так:

```bash
godot --headless --path spin-target --export-release "Android" "SpinTarget-Android.apk"
```

### RPM для ОС Аврора

RPM-пакет для ОС Аврора собирается в настроенном окружении Aurora SDK / Platform SDK. В качестве исходников используется Godot-проект из каталога `spin-target/`.

Общий порядок сборки:

1. Подготовьте окружение сборки под ОС Аврора.
2. Откройте или импортируйте проект из каталога `spin-target/`.
3. Соберите пакет для архитектуры `armv7hl`.
4. Проверьте, что результат имеет расширение `.rpm`.
5. Для релиза переименуйте файл в понятный формат:

```text
SpinTarget-demo-2026-05-18-AuroraOS.rpm
```

Актуальный RPM для демонстрационного релиза приложен к GitHub Release `demo-2026-05-18`.

### Windows для локальной проверки

В проекте есть пресет **Windows Desktop**. Его можно использовать для быстрой проверки игры на ПК:

```bash
godot --headless --path spin-target --export-release "Windows Desktop" "SpinTarget.exe"
```
