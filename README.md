# Spin Target (Аврора)

Учебный проект игры в жанре **one hit** для операционной системы **Аврора**, созданный на игровом движке **Godot**.

## О проекте
Это простая аркадная игра, с основной механикой метания ножей во вращающуюся мишень. Проект разработан в рамках изучения разработки под ОС Аврора и возможностей Godot.
